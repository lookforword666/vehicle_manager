# vehicle_manager
北京卫戍区派车管理系统软件项目
介绍
  北京卫戍区外包派车管理软件项目，由我负责，其他的看介绍

软件架构
  技术：Springboot、mysql、Springmvc、mybatis plus

安装教程
  下载vehicle_manager_2.0.zip项目（mac压缩的里面会有一个_MACOSX类似文件夹 将其删掉即可）
  使用idea/eclipse导入项目
  导入数据库脚本和更改数据库配置
  打包jar 部署到服务器上即可使用 部署很简单就不介绍如何部署了
